# 🎓 CGPA to Package Predictor

A production-ready Machine Learning web application designed to estimate a student's expected placement package 💼 based on their CGPA.

This project demonstrates the end-to-end implementation of a regression model integrated with an interactive web interface using **Streamlit**, focusing on usability, visualization, and real-world relevance.

---

## 🚀 Key Features

* 🎯 Real-time prediction of placement package based on CGPA
* 📊 Interactive dashboard with clean and intuitive UI
* 📈 Dynamic visualization of CGPA vs Package relationship
* 💡 Insight-driven feedback based on prediction output
* ⚡ Lightweight and fast deployment using Streamlit

---

## 🧠 Machine Learning Overview

* **Algorithm:** Linear Regression
* **Problem Type:** Supervised Learning (Regression)
* **Input Feature:** CGPA (Range: 0 – 10)
* **Target Variable:** Placement Package (LPA)
* **Libraries:** scikit-learn, NumPy

The model learns the relationship between academic performance (CGPA) and placement outcomes, enabling quick and interpretable predictions.

---

## 🛠️ Technology Stack

* **Programming Language:** Python
* **Frontend Framework:** Streamlit
* **Data Processing:** NumPy
* **Visualization:** Matplotlib
* **Model Serialization:** Joblib

```

```

```

```

## 📊 Application Workflow

1. User inputs CGPA through the UI
2. Input is passed to the trained regression model
3. Model generates predicted package value
4. Result is displayed along with visual insights
5. Graph illustrates the overall trend and prediction position
